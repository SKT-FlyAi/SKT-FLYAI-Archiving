# 에지 검출
## edge란
디지털 영상의 밝기의 변화량이 큰 지점(경계)

## 1차 미분 마스크
함수의 순간 변화율을 구하는 계산 과정을 의미   
에지가 화소의 밝기가 급격히 변하는 부분이기 때문에 함수의 변화율을 취하는 미분 연산을 이용해서 에지 검출 가능
